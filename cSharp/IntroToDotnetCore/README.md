# MVC

## What is an MVC?
* Model
* View
* Controller

## Controller responsibility?
* Incoming Request -> response(string, html, json, redirect)
* Controller Class (inherit from ASPCore)

## View responsibility?
* UI Layer
* Razor (.cshtml)
* Pass data -> .cshtml: ViewBag

## Model responsibility?
* Data layer
* Logic layer
* Interaction to DB

## Models
